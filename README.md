# TypeScript Essentials

Entrenamiento de TypeScript esencial como preparación para Patrones de Diseño.

Código puro + Tests con Jest. Sin HTML/CSS.

## Qué cubre

- **Día 1:** Setup profesional (NPM, TypeScript, Jest) + Types básicos
- **Día 2:** Interfaces, Type Aliases, Union/Intersection types, Enums, Type Guards
- **Día 3:** Generics, Clases genéricas, Abstract classes, Polimorfismo
- **Día 4:** Proyecto integrador (Task Manager) + Refactoring JS→TS

## Stack

- TypeScript 5
- Jest + ts-jest
- Node.js

## Comandos

```bash
npm install          # Instalar dependencias
npm run build        # Compilar TS → JS
npm test             # Correr todos los tests
npm run test:watch   # Tests en modo watch
```

## Estructura

```
src/
├── day-1/   → Types básicos, arrays, tuples, objects
├── day-2/   → Interfaces, type aliases, enums, type guards
├── day-3/   → Generics, generic classes, abstract classes
└── day-4/   → Task Manager (integrador) + UserManager (refactoring JS→TS)
```
